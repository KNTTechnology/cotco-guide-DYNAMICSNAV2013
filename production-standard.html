คู่มือ 03 · เมนูมาตรฐานการผลิต

มาตรฐานการผลิต — ตัวเลขที่ทำให้ระบบคำนวณเวลาเองได้

เมนู มาตรฐานการผลิต (Routing Standard) เก็บ “เวลามาตรฐาน” ของแต่ละสินค้าบนแต่ละเครื่องจักร เช่น Cycle time และเวลาเปลี่ยนงาน — ตัวเลขชุดนี้คือสิ่งที่ทำให้หน้าวางแผนการผลิตคำนวณเวลาเริ่ม/สิ้นสุดให้อัตโนมัติ

หลักการ

1 แถว = 1 สินค้า × 1 เครื่องจักร

สินค้าเดียวกันแต่คนละเครื่อง มีมาตรฐานคนละแถว

เวลาในหน้าจอกรอกเป็น “นาที”

ทุกช่องเวลา (Cycle time, เวลาเปลี่ยนงาน) กรอกและแสดงเป็นนาที — ระบบเก็บเป็นวินาทีให้เองเบื้องหลัง ไม่ต้องแปลง

ทำอะไรได้

สิ่งที่ทำได้ในเมนูนี้

มีทั้งเพิ่มทีละรายการ และนำเข้าเป็นชุดด้วย Excel

ค้นหา / กรอง

หารายการ

ค้นด้วย Item No / ชื่อสินค้า, กรองตามเครื่องจักร, โรงงาน (P1/P2) และแสดงรายการที่ปิดใช้งานได้

เพิ่ม / แก้

ทีละรายการ

กด “เพิ่มมาตรฐาน” เพื่อสร้างเอง หรือกดไอคอนดินสอเพื่อแก้แถวเดิม

Excel

นำเข้า / ส่งออก

ส่งออกรายการเป็น .xlsx ไปแก้ แล้วนำเข้ากลับทีเดียวหลายรายการ (อัปเดต/เพิ่มอัตโนมัติ)

ข้อมูล

ข้อมูลที่แต่ละมาตรฐานเก็บ

ช่องที่มีผลต่อการคำนวณเวลาในแผนโดยตรง

ช่องความหมายใช้ทำอะไร
Item No (จำเป็น)รหัสสินค้าคู่กับเครื่องจักร ใช้ระบุว่าเป็นมาตรฐานของอะไร
ชื่อสินค้าคำอธิบายสินค้าแสดงในตารางรายการ
ขนาด (S / M / L)คลาสขนาดของงานเลือกว่าจะใช้เวลาเปลี่ยนงาน “→ S/M/L” ค่าไหน
มิติ (หนา / กว้าง / ยาว)ขนาดแผ่น (มม.)ใช้เลือกเวลาเปลี่ยนงานเมื่อความหนา/ยาวเปลี่ยน
เครื่องจักร (จำเป็น)ศูนย์เครื่องจักรที่ใช้มาตรฐานนี้เลือกจากรายการเครื่องที่มีอยู่
Cycle time (นาที/ชิ้น)เวลาผลิตต่อชิ้น — ตัวเลขหัวใจเวลาผลิต = จำนวน × Cycle time
น้ำหนัก/ชิ้น (กก.)น้ำหนักเฉลี่ยต่อชิ้นใช้เป็นน้ำหนักสำรองในแผนเมื่อ NAV ไม่มีค่าน้ำหนัก
เปลี่ยน → S / M / Lเวลาเปลี่ยนงานไปคลาสขนาดนั้นใช้เป็น Setup เริ่มต้นเมื่อขนาดเปลี่ยนระหว่างงาน
เปลี่ยนความหนา / ความยาวเวลาเปลี่ยนงานเมื่อหนา/ยาวเปลี่ยนใช้เป็น Setup เริ่มต้นตามการเปลี่ยนมิติ
Setup (เฉพาะงานแปรรูป)ค่า setup ค่าเดียวใช้กับแถวงานแปรรูปเท่านั้น (ปกติเว้นว่าง)
โรงงาน (P1 / P2)ขอบเขตโรงงานของมาตรฐานคุมว่าใครแก้ได้ (ตามสิทธิ์โรงงาน)
สถานะใช้งาน / ปิดใช้งานปิดใช้งาน = ซ่อนจากการใช้งาน (เปิดกลับได้)

หมายเหตุ: ช่อง Sizing S/M/L เป็นข้อมูลอ้างอิงเท่านั้น — ระบบจัดตารางเวลาไม่ได้ใช้ค่านี้

เชื่อมโยง

มีผลกับหน้าวางแผนอย่างไร

นี่คือเหตุผลว่าทำไมบางแถวในแผนถึงแก้เวลาเองไม่ได้

มี Cycle time > 0 → แถวถูกล็อกเวลา

ถ้าสินค้าในแถวแผนมีมาตรฐานที่ Cycle time > 0 → แถวนั้นเป็นแบบ “มี Routing” ระบบคุมเวลาเริ่ม/สิ้นสุดให้ (แก้ได้แค่ Setup)

จำนวนยิ่งมาก เวลายิ่งนาน

เวลาผลิต = จำนวน × Cycle time — เพิ่มจำนวนหรือ Cycle สูงขึ้น เวลาสิ้นสุดก็เลื่อนออกไป

เวลาเปลี่ยนงาน = Setup เริ่มต้น

ระบบดูการเปลี่ยนจากงานก่อนหน้า: ขนาดเปลี่ยน→ใช้ “เปลี่ยน→ขนาดใหม่”, หนาเปลี่ยน→“เปลี่ยนความหนา”, ยาวเปลี่ยน→“เปลี่ยนความยาว” (ผู้วางแผน override เองได้)

ไม่มีมาตรฐาน → ตกไปบล็อก 2 ชม.

สินค้าที่ไม่มีมาตรฐาน (หรือ Cycle = 0) จะกลายเป็นแถว “ไม่มี Routing” — ผู้วางแผนคุมเวลาเอง และได้บล็อกมาตรฐาน 2 ชม.

Excel

นำเข้าด้วย Excel

วิธีหลักสำหรับอัปเดตหลายรายการพร้อมกัน

1

ส่งออกไฟล์

กด “ส่งออก Excel” ได้ไฟล์ .xlsx ของรายการปัจจุบันไปแก้

2

นำเข้ากลับ

กด “นำเข้า Excel” เลือกไฟล์ .xlsx ระบบจะอัปเดตของเดิม + เพิ่มของใหม่ให้อัตโนมัติ

3

ดูผลลัพธ์

ระบบสรุป “ทั้งหมด · เพิ่ม · อัปเดต · ผิดพลาด” พร้อมรายการแถวที่ผิดให้แก้

·

ข้อกำหนดไฟล์นำเข้า

ต้องเป็นไฟล์ .xlsx — และมีคอลัมน์บังคับ: Item No, เครื่องจักร, โรงงาน
สูงสุด 5,000 แถวต่อไฟล์ — เกินกว่านั้นระบบจะไม่รับ
นำเข้าต้องมีสิทธิ์ “เพิ่ม” และ “แก้ไข” — เพราะเป็นการทั้งเพิ่มและอัปเดต
แถวที่อยู่นอกโรงงานของคุณจะถูกข้าม — แถวอื่นยังนำเข้าได้ตามปกติ และรายงานแถวที่ผิดให้
สิทธิ์ · สถานะ

สิทธิ์และสถานะ

ปุ่มจะขึ้นเฉพาะเมื่อคุณมีสิทธิ์และแถวอยู่ในโรงงานของคุณ

สถานะรายการ

การลบเป็นการ “ปิดใช้งาน” ไม่ใช่ลบจริง — เปิดกลับได้ทีหลัง

ใช้งาน (Active)·ปิดใช้งาน (Inactive)

ติ๊ก “แสดงรายการที่ปิดใช้งาน” เพื่อค้นและกดเปิดใช้งานกลับ · ทุกการเปลี่ยนแปลงถูกบันทึกประวัติ (audit)

ตรวจก่อนบันทึก

กรอกไม่ครบหรือผิด ระบบจะไฮไลต์ช่องสีแดง

Item No / เครื่องจักร ต้องระบุ โรงงานต้องเป็น P1 / P2 ตัวเลขต้องเป็น 0 ขึ้นไป
